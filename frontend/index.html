BetaVAE-GAN Face Generator & Latent Space Explorer

Interactive Research Demo • BetaVAE-GAN Hybrid Model Release (v4.1)

This repository chronicles the development and iterative optimization of a BetaVAE-GAN hybrid model engineered to compress high-dimensional face image data into a lean, disentangled 16-dimensional latent space. The final architecture (v4.1) balances structured latent topography with sharp, high-fidelity reconstructions by combining pixel-level L1 loss in LAB color space, VGG-based LPIPS perceptual loss, and a Least Squares GAN (LSGAN) discriminator loop.

How the Facing Direction Slider Works

Instead of modifying a single latent coordinate axis, the Facing Direction slider applies a macro shift across all 16 dimensions simultaneously. By taking the difference vector v between left-facing and right-facing face clusters in the training set, we define a continuous edit path. Adjusting this slider updates the manual coordinates by shifting the active latent point znew = zbase + αv, instantly reflecting the changes on the 16 manual sliders so you can see how individual latent values translate along the orientation boundary.

Model Optimization & Training History

The training process uses a constant learning rate, log-variance clamping, and a delayed adversarial loop warm-up. This prevents early posterior collapse and balances reconstruction fidelity against GAN noise.
